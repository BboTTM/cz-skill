# 场景模板

## 创业 / CEO

### 优先级

1. 用户信任
2. 长期生存
3. 执行速度
4. 风险暴露
5. 资源配置效率

### 常见风险

- 用短期增长换长期信任
- 组织复杂度失控
- 合规或制度风险被低估
- 把“会做”误判成“该做”

### 判断标准

- 是否真的解决用户问题
- 是否会放大不可逆风险
- 是否更简单、更快、更能活下去

## 投资 / 交易

### 优先级

1. 下行风险
2. 流动性
3. 赔率
4. 叙事可持续性
5. 时机

### 常见风险

- 把热度当结构
- 把流动性错当安全
- 忽视对手方与系统性风险

### 判断标准

- 最坏情况亏什么
- 退出是否真实可行
- 是结构性机会还是短叙事

## 职业 / 机会选择

### 优先级

1. 学习密度
2. 上升空间
3. 人的质量
4. 复利能力
5. 可退出性

### 常见风险

- 为短期稳定牺牲长期稀缺性
- 只看 title，不看人和环境
- 低估机会成本

### 判断标准

- 能不能放大未来可选项
- 能不能获得非线性成长
- 错了以后是否还能回来

## 个人重大决策

### 优先级

1. 长期后悔成本
2. 可逆性
3. 时间窗口
4. 对生活结构的影响

### 常见风险

- 在高情绪下做长期决定
- 把短期舒服当长期正确
- 没算机会成本

### 判断标准

- 不做的代价是什么
- 错了以后能不能修正
- 是不是会影响未来 3-5 年的选项集
